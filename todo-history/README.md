# Todo History

Folder ini berisi catatan masalah UI/UX dan bug yang ditemukan selama development BidanCRM. Setiap file merepresentasikan satu masalah spesifik dengan daftar todo checkbox untuk tracking perubahan.

## Format Penamaan File

```
{YYYY-MM-DD}-{title-problem}.md
```

## Cara Menggunakan

1. Setiap masalah baru yang ditemukan dibuatkan file `.md` baru dengan format penamaan di atas
2. Todo checkbox digunakan untuk tracking progress perubahan
3. Setelah semua checkbox selesai, file bisa diarsipkan atau ditandai sebagai `[DONE]` di judul

## Status Legend

- [ ] Belum dikerjakan
- [x] Sudah selesai
- [~] Sedang dikerjakan / partial

## Daftar Masalah

Lihat file-file `.md` di folder ini untuk detail masing-masing masalah.
